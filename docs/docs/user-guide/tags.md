# Command: tags

## When To Use It?

   * Want to see a list of tags used on a pelican site

## Examples

### List tags

```
$ lapis tags

[1] bird
[1] cliff
[1] crane
[1] ocean
[2] photography
[1] shore
[2] water
```

### List tags ordered by count

```
$ lapis tags -c 

[2] photography
[2] water
[1] bird
[1] crane
[1] shore
[1] ocean
[1] cliff
```
